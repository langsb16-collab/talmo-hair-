## 페이지별 URL 구조

| 페이지 | URL | 설명 |
|--------|-----|------|
| 메인 페이지 | `/` | 플랫폼 소개 및 CTA |
| AI 탈모 진단 | `/diagnosis` | 사진 업로드 및 AI 분석 |
| 가상 시뮬레이션 | `/simulation` | 수술 후 결과 시뮬레이션 |
| 병원 매칭 | `/hospitals` | 병원 비교 및 상담 신청 |
| 후기 & 데이터 | `/reviews` | 실제 수술 후기 및 통계 |

## 사용자 플로우 (User Flow)

### 플로우 1: 진단 → 시뮬레이션 → 병원 매칭
```
홈 페이지
  ↓ (무료 AI 진단 시작)
AI 진단 페이지
  ↓ (사진 업로드 & 분석)
진단 결과 확인
  ↓ (수술 결과 시뮬레이션 보기)
시뮬레이션 페이지
  ↓ (옵션 선택 & 생성)
시뮬레이션 결과 확인
  ↓ (적합한 병원 찾기)
병원 매칭 페이지
  ↓ (병원 선택)
상담 신청
```

### 플로우 2: 시뮬레이션 체험 → 진단 → 병원
```
홈 페이지
  ↓ (시뮬레이션 체험)
시뮬레이션 페이지
  ↓ (결과 확인 후 정확한 진단 필요)
AI 진단 페이지
  ↓ (맞춤 진단 후)
병원 매칭 페이지
```

### 플로우 3: 후기 확인 → 시뮬레이션 → 병원
```
홈 페이지
  ↓ (후기 메뉴)
후기 페이지
  ↓ (실제 사례 확인 후 관심)
시뮬레이션 페이지
  ↓ (내 경우 확인)
병원 매칭 페이지
```

## 향후 추가 예정 페이지

### 인증 & 사용자
- `/login` - 로그인
- `/signup` - 회원가입
- `/mypage` - 마이페이지
  - `/mypage/simulations` - 내 시뮬레이션 기록
  - `/mypage/consultations` - 상담 내역
  - `/mypage/subscriptions` - 구독 관리

### 병원 파트너
- `/partner/login` - 병원 관리자 로그인
- `/partner/dashboard` - 병원 대시보드
- `/partner/consultations` - 상담 예약 관리
- `/partner/reviews` - 후기 관리
- `/partner/analytics` - 통계 및 분석

### 정보 & 고객지원
- `/guide` - 탈모 가이드
- `/pricing` - 비용 안내
- `/faq` - 자주 묻는 질문
- `/contact` - 고객센터
- `/about` - 회사 소개
- `/terms` - 이용약관
- `/privacy` - 개인정보처리방침

### 결제 & 금융
- `/payment` - 결제 페이지
- `/finance` - 의료 금융 안내
- `/insurance` - 재수술 보증 상품

## 반응형 디자인

모든 페이지는 다음 디바이스에 최적화되어 있습니다:
- 📱 모바일 (320px~767px)
- 📱 태블릿 (768px~1023px)
- 💻 데스크톱 (1024px+)

## 접근성

- ✅ 시맨틱 HTML5 마크업
- ✅ ARIA 레이블 (향후 추가)
- ✅ 키보드 네비게이션 지원
- ✅ 고대비 색상 (WCAG AA 준수)
- ✅ 스크린 리더 호환 (향후 개선)

## SEO 최적화 (향후)

각 페이지별 메타 태그:
- `<title>` - 페이지별 고유 제목
- `<meta name="description">` - 페이지 설명
- `<meta property="og:*">` - Open Graph 태그
- `<meta name="twitter:*">` - Twitter 카드
- `<link rel="canonical">` - 정규 URL
